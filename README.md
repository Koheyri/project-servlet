# Project Servlet

Веб-приложение на Java Servlets + JDBC для выполнения базовых CRUD-операций. Базовая реализация веб-приложения с использованием Servlet API и прямого доступа к базе данных.

## Описание проекта

Проект демонстрирует работу с Java Servlets, обработку HTTP-запросов и выполнение CRUD-операций через JDBC без использования фреймворков. Предназначен для понимания основ веб-разработки на Java.

**Технологический стек:**
- Java
- Servlet API
- JDBC
- Maven
- Apache Tomcat
- CSS

## Цель проекта

| Аспект | Описание |
|--------|----------|
| Что решает | Демонстрация работы с Java Servlets, обработка HTTP-запросов, выполнение CRUD-операций через JDBC |
| Для кого | Учебный проект для понимания основ веб-разработки на Java без использования фреймворков |

## Функциональные требования

### User Stories

| ID | Описание | Приоритет |
|----|----------|-----------|
| US-1 | Инициализация приложения для подготовки БД к работе | High |
| US-2 | Выполнение бизнес-логики через веб-интерфейс | High |
| US-3 | Создание, чтение, обновление и удаление данных | High |
| US-4 | Чистая структура servlet'ов для поддержки кода | Medium |

### Use Cases

- Инициализация базы данных (InitServlet)
- Выполнение CRUD-операций (LogicServlet)

## Модель данных

### Компоненты

| Компонент | Назначение |
|-----------|------------|
| InitServlet | Инициализация БД, создание таблиц, загрузка начальных данных |
| LogicServlet | Обработка бизнес-логики, CRUD-операции |
| JDBC Connection | Управление подключением к базе данных |
